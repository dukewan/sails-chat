# Sails-chat

An online chat room built with Sails.js which demonstrates the key features of Sails:

+ Blueprints
+ Walterline
+ Socket.io

## Set up

```
npm install sails@beta -g
npm install
sails lift
// visit http://localhost:1337/slack
```
